Weather forecast

This application can do everything what some weather app should do. When you open it for the first time you need to choose city and it will be save. Of course you can change it later, but app remember your choice. On the main screen you can see basic info, but you can switch on advance screen for more infromation. App is for mobile, but you can check it on larger device.
